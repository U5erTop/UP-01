# Темы для учебных проектов

## 1. Система управления книжным магазином

**Описание:** Создать приложение для управления сетью книжных магазинов.

**Что нужно сделать студентам:**
- Реализовать каталог книг с авторами, жанрами и издательствами
- Создать систему управления складом и учет остатков
- Разработать функцию оформления покупок (выбор книг, расчет суммы)
- Вести данные постоянных покупателей
- Создать отчеты по популярным авторам, жанрам и книгам
- Добавить функцию поиска книг по названию или автору
- Реализовать систему скидок для постоянных покупателей
